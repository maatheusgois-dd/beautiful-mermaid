Data security

Your diagrams never leave your browser.

They're only stored in the URL and your browser's local storage.

This is a fully open source, client-side app that works offline after the first load.

Your code is rendered entirely on-device by the Beautiful Mermaid engine. External actions you trigger (Copy URL, Export PNG/SVG) run locally — nothing is ever uploaded.

We use Firebase Analytics to collect anonymous usage metadata (page views, feature clicks). Diagram content is never included.
